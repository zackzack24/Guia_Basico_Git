
# Ramificaçôes ou Branches

## Criar uma nova branch
A branch padrão é a branch MAIN
```bash
git branch <nome-da-branch>
```

## Mudar a branch atual
Mudar a Branch pode servir para isolar o trabalho em diferentes funcionalidades ou correções.
```bash
git checkout <nome-da-branch>
```
Ou
```bash
git switch <nome-da-branch>
```

## Listar suas Branches
```bash
git branch
```

## Juntar ou Mesclar Branches
Ajuda a trabalhar de forma mais organizada
```bash
git merge <nome-da-branch>
```

## Deletar uma Branch
```bash
git branch -d <nome-da-branch>
```
