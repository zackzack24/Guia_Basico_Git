# Comandos Básicos do Git

## Inicializar um repositório
Nunca faça esse comando mais de uma vez no mesmo repositório!!
```bash
git init
```

## Clonar um repositório
Criar uma cópia exata de um repositório Git existente, na sua máquina local
```bash
git clone <url>
```

## Verificar Status
Vai mostrar um breve resumo do que está acontecendo no seu código
```bash
git status
```

## Adicionar arquivos ao staging
```bash
git add <arquivo>
git add .  ## adiciona todos os arquivos
```

## Fazer ou Nomear commit
Veja a convenção de nomes de commits aqui ***[Strong emphasized text](https://www.conventionalcommits.org/en/v1.0.0/)***
```bash
git commit -m "mensagem do seu commit"
```

## Adicionar ao Repositório
Envia as alterações (commits) do seu repositório local para um repositório remoto
```bash
git push 
```

## Ver Histórico
Exibe o histórico de commits de um repositório
```bash
git log
```
